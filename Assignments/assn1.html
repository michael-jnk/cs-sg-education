What is HTML & CSS

HTML, CSS, and JavaScript are the three components that make up websites today. HTML is the base components & structure of a website, CSS creates the look of all the elememts on it, and JavaScript adds extra function & interactability to HTML elements.
This website will go over the basic structure of HTML and CSS files, as well as showing some common elements in each.

HTML File Structure

A typical HTML file begins with two tags - <!DOCTYPE html> and <html lang="en"> , which simply state the file is an HTML file written in English, but the rest of the file is contained inside the <head> and <body> tags.

<head>

Within the <head> tag is all the metadata & information about the website that isn't displayed on the page. To demonstrate, I'll show the head of this file, and explain what each part means: 

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link href="style1.css" rel="stylesheet">
    <title>Intro to HTML & CSS - MJ</title>
</head>
